# Proyecto
Este repositorio contiene nuestro proyecto sobre árboles generales y binarios.

# ¿Por qué se usó una lista para definir un árbol generalizado?
Esto es porque las listas permiten guardar una infinidad de datos, en este caso árboles. Entonces, primero se puso 
para cuando es un árbol vacío, y luego se puso que se haga un nodo a que tenga como hijos a todos los árboles que
están en la lista.
